== README

This simple application that lets you:

	* Sign up and create an account

	* Log in to your account plus logout and has a simple authentication system (no authentication gem used)

	* Search artists from Last.fm

	* Show similar artists from the searched artist

	* Show user's searched history

To run the application on your system, you have to paste
your api key and api secret provided from your account at
Last.fm in the config/secrets.yml file
